# BootSync 이용약관 초안

최종 수정일: 2026-03-16

이 문서는 BootSync 공개 전 검토용 초안입니다. 실제 운영 주체, 도메인, 문의처를 채운 뒤 확정해야 합니다.

## 1. 목적

이 약관은 BootSync 서비스 이용과 관련하여 운영자와 이용자 사이의 권리, 의무, 책임사항을 정하는 것을 목적으로 합니다.

## 2. 서비스 내용

BootSync는 국비지원 IT 교육과정 수강생을 위해 아래 기능을 제공합니다.

- 개인 출결 기록
- 예상 훈련장려금 계산
- 과정 현황 확인
- 학습 노트 저장과 검색
- 복구 이메일 기반 계정 복구 보조

## 3. 이용자 계정

이용자는 본인 정보를 바탕으로 계정을 생성하고 이용해야 합니다.

이용자는 다음 사항을 지켜야 합니다.

- 타인의 정보를 도용하지 않기
- 비밀번호를 안전하게 보관하기
- 자신의 계정을 타인과 공유하지 않기

## 4. 서비스 이용 제한

운영자는 아래 경우 서비스 이용을 제한할 수 있습니다.

- 타인의 계정을 무단 사용한 경우
- 다른 이용자의 데이터에 부정 접근을 시도한 경우
- 서비스 보안을 해치는 행위를 한 경우
- 법령 또는 공공질서에 반하는 목적으로 사용한 경우

## 5. 데이터 소유와 책임

이용자가 입력한 출결 기록, 학습 노트, 과정 정보는 해당 이용자 본인의 데이터로 취급합니다.

다만 이용자는 아래 사항을 이해하고 사용해야 합니다.

- BootSync의 장려금 계산은 참고용 예상값입니다.
- 실제 지급 기준은 학원, HRD-Net, 행정 담당자의 공지를 우선합니다.
- 이용자가 직접 입력하지 않은 과거 날짜는 자동 결석 처리되지 않을 수 있습니다.

## 6. 장려금 계산 안내

BootSync는 저장된 출결 기록과 과정 정보를 기준으로 예상 장려금을 계산합니다.

현재 기본 계산 원칙:

- 1일 지급액 × 지급 반영 일수
- 지각 3회는 결석 1회 환산
- 조퇴 3회는 결석 1회 환산

이 계산 결과는 실제 지급 금액과 차이가 있을 수 있습니다.

## 7. 서비스 중단과 변경

운영자는 서비스 개선, 점검, 장애 대응을 위해 일부 기능을 변경하거나 일시 중단할 수 있습니다.

중요한 변경이 있는 경우 가능한 범위에서 사전에 안내합니다.

## 8. 계정 삭제

이용자는 언제든지 계정 삭제를 요청할 수 있습니다.

삭제 요청 후에는:

- 계정 상태가 삭제 대기 상태로 전환될 수 있고
- 현재 세션이 종료되며
- 유예 기간 뒤 실제 데이터 삭제가 진행될 수 있습니다

세부 절차는 [ACCOUNT_DELETION_AND_RECOVERY_POLICY.md](/C:/B_Recheck/docs/policies/ACCOUNT_DELETION_AND_RECOVERY_POLICY.md)를 따릅니다.

## 9. 면책

운영자는 아래 사항에 대해 책임을 제한할 수 있습니다.

- 이용자가 입력한 잘못된 출결 정보로 인한 오차
- 학원 또는 HRD-Net의 실제 지급 기준과의 차이
- 이용자의 비밀번호 관리 소홀로 발생한 문제
- 천재지변, 장애, 외부 서비스 장애로 인한 불가피한 중단

## 10. 문의

공개 전 아래 정보를 실제 운영값으로 채워야 합니다.

- 운영 주체명
- 문의 이메일

예시:

- 운영 주체: `<YOUR_ORG_NAME>`
- 문의 메일: `<SUPPORT_EMAIL>`
